Selected Projects

PDF Reader with Ollama

Python, Flask, pypdf, Ollama, local AI workflow

Built a local Flask application that uploads a PDF, extracts readable text, and uses Ollama to answer user questions from the document. The app supports asking questions in English and getting responses in Telugu, Tamil, Kannada, Malayalam, Hindi, or English.

The application includes browser PDF preview, extracted text display, model selection, follow-up chat with the same PDF, and Telugu audio generation for responses. It was designed as a practical local-AI assistant for multilingual document understanding.

Indian Passport Photo Helper

Python, Flask, OpenCV, Pillow, NumPy

Built a Python web application to transform a portrait photo into a passport-style 2 x 2 image aligned to Indian passport-photo expectations. The app performs face detection, auto-straightening, crop preparation, background cleanup, and exact JPEG export.

It also includes manual controls for zoom, face position, and rotation, plus a compliance-style checklist for face ratio, centering, sharpness, lighting, background quality, and final file-format validation such as JPEG mode, size, and DPI metadata.
