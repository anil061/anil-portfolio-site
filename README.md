## Publish To GitHub Pages

1. Create a new GitHub repository, for example `anil-portfolio-site`.
2. Push the contents of this folder to the repository.
3. In GitHub:
   - open the repository
   - go to `Settings`
   - open `Pages`
   - under `Build and deployment`, choose `Deploy from a branch`
   - select branch `main`
   - select folder `/ (root)`
4. Save the settings and wait for GitHub Pages to publish the site.

If you want the site at a custom URL like `https://anil061.github.io/anil-portfolio-site/`, this folder is already ready for that style of deployment.
